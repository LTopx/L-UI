## API

### Modal Props

| 属性             | 描述                 | 类型                 | 默认值    |
| ---------------- | -------------------- | -------------------- | --------- |
| className        | 节点类名             | `string`             | -         |
| overlayClassName | 遮罩层类名           | `string`             | -         |
| width            | 对话框宽度           | `string` \| `number` | `32.5rem` |
| maskClosable     | 是否可点击遮罩层关闭 | `boolean`            | true      |
| title            | 自定义标题           | `React.ReactNode`    | Title     |
| footer           | 自定义页脚           | `React.ReactNode`    | -         |
| cancelText       | 自定义取消按钮文字   | `React.ReactNode`    | Cancel    |
| okText           | 自定义确认按钮文字   | `React.ReactNode`    | Ok        |
| open             | 对话框是否可见       | `boolean`            | -         |
| onOk             | 点击“确认”按钮回调   | `() => void`         | -         |
| onClose          | 对话框关闭时的回调   | `() => void`         | -         |
