## 基本情况

LX-UI 是一个 React 开源 UI 库，基于 headless 风格的 Radix UI，采用了 Tailwind CSS 框架，使用 TypeScript 编写，支持 SSR。

目前市面上有许多优秀的 UI 库，比如功能完备的 Ant Design、Chakra UI，headless 风格的 Radix UI、Headless UI，还有异常火爆的 shadcn/ui 等等，它们都非常优秀。

在开发 L-GPT 时，考虑了上述选项之后，一是为了减轻心智负担，二是能够具有一定的样式特色，于是决定自己写一个 UI 库，然后就有了 LX-UI。
