# 快速开始

## LX-UI

`LX-UI` 是基于 [Radix UI](https://www.radix-ui.com) 和 [Tailwind CSS](https://tailwindcss.com/) 二次封装开发的一套 React 组件库。

项目创建的目的是为了解决个人开发项目 [L-GPT](https://gpt.ltopx.com) 的 UI 统一问题，因此前期会比较简陋 🤣🤣。后续会逐步迭代完善。

## 特性

- 💎 基于 [Radix UI](https://www.radix-ui.com)，丰富和拓展了部分组件
- 🔥 使用了 [Tailwind CSS](https://tailwindcss.com)，便于后续样式定制
- 🎉 支持 Typescript
- 📱 支持响应式
