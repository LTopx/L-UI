### Controlled mode

<Code component={<Checked />}>
<Code.Description>
Set `checked` to put the switch in controlled mode.
</Code.Description>
```tsx filename="Switch.tsx" copy
import { Switch } from "@ltopx/lx-ui";

export default function () {
  const [checked, setChecked] = React.useState(false);
  
  return <Switch checked={checked} onChange={setChecked} />;
}
```
</Code>

## API

### Switch Props

| Name           | Description                               | Type                         | Default |
| -------------- | ----------------------------------------- | ---------------------------- | ------- |
| className      | Additional css class                      | `string`                     | -       |
| thumbClass     | Middle dot class                          | `string`                     | -       |
| checked        | Whether it is open                        | `boolean`                    | -       |
| defaultChecked | Whether to open by default                | `boolean`                    | -       |
| onChange       | Callback when switching the toggle state. | `(checked: boolean) => void` | -       |
