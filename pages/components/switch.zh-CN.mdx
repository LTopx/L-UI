### 受控模式

<Code component={<Checked />}>
<Code.Description>
设置 `checked`，让开关处于受控模式
</Code.Description>
```tsx filename="Switch.tsx" copy
import { Switch } from "@ltopx/lx-ui";

export default function () {
  const [checked, setChecked] = React.useState(false);
  
  return <Switch checked={checked} onChange={setChecked} />;
}
```
</Code>

## API

### Switch Props

| 属性           | 描述                 | 类型                         | 默认值 |
| -------------- | -------------------- | ---------------------------- | ------ |
| className      | 节点类名             | `string`                     | -      |
| thumbClass     | 中间圆点类名         | `string`                     | -      |
| checked        | 受控模式，是否打开   | `boolean`                    | -      |
| defaultChecked | 默认打开状态         | `boolean`                    | -      |
| onChange       | 切换开关状态时的回调 | `(checked: boolean) => void` | -      |
