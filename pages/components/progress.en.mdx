## API

### Progress Props

| Name      | Description          | Type                                                         | Default   |
| --------- | -------------------- | ------------------------------------------------------------ | --------- |
| className | Additional css class | `string`                                                     | -         |
| type      | Progress type        | `default` \| `primary` \| `success` \| `warning` \| `danger` | `default` |
| size      | Progress size        | `sm` \| `base` \| `lg`                                       | `base`    |
| value     | progress             | `number`                                                     | 0 (0-100) |
