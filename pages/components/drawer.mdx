### 基础用法

<Code 
  description='点击按钮，抽屉从右侧滑出。点击遮罩层或关闭按钮可关闭抽屉。' 
  component={<Basic />}
>
```tsx filename="Drawer.tsx" copy
import React from "react";
import { Drawer, Button } from "@ltopx/lx-ui";

export default function () {
  const [open, setOpen] = React.useState(false);
  return (
    <>
      <Button type="primary" onClick={() => setOpen(true)}>
        Drawer
      </Button>
      <Drawer title="这是一个title" open={open} onClose={() => setOpen(false)}>
        <div>Drawer Content</div>
      </Drawer>
    </>
  );
}
```
</Code>

### 弹出位置

<Code 
  description='自定义抽屉弹出位置。目前支持从左右两侧弹出' 
  component={<Placement />}
>
```tsx filename="Drawer.tsx" copy {20,28}
import React from "react";
import { Drawer, Button } from "@ltopx/lx-ui";

export default function () {
  const [openLeft, setOpenLeft] = React.useState(false);
  const [openRight, setOpenRight] = React.useState(false);
  
  return (
    <>
      <div className="flex gap-2">
        <Button type="primary" onClick={() => setOpenLeft(true)}>
          Left Drawer
        </Button>
        <Button type="primary" onClick={() => setOpenRight(true)}>
          Right Drawer
        </Button>
      </div>
      <Drawer
        title="这是一个title"
        placement="left"
        open={openLeft}
        onClose={() => setOpenLeft(false)}
      >
        <div>Drawer Content</div>
      </Drawer>
      <Drawer
        title="这是一个title"
        placement="right"
        open={openRight}
        onClose={() => setOpenRight(false)}
      >
        <div>Drawer Content</div>
      </Drawer>
    </>
  );
}
```
</Code>

## API

### Drawer Props

| 属性             | 描述                     | 类型                 | 默认值  |
| ---------------- | ------------------------ | -------------------- | ------- |
| className        | 内容区域类名             | `string`             | -       |
| overlayClassName | 遮罩层类名               | `string`             | -       |
| title            | 自定义标题               | `React.ReactNode`    | Title   |
| placement        | 抽屉弹出方向             | `left` \| `right`    | `right` |
| open             | 是否显示抽屉             | `boolean`            | -       |
| width            | 抽屉显示宽度             | `number` \| `string` | 378     |
| onClose          | 关闭抽屉时触发的回调函数 | `() => void`         | -       |
