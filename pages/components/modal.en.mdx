## API

### Modal Props

| Name             | Description                                | Type                 | Default   |
| ---------------- | ------------------------------------------ | -------------------- | --------- |
| className        | Additional css class                       | `string`             | -         |
| overlayClassName | Mask layer class                           | `string`             | -         |
| width            | Modal width                                | `string` \| `number` | `32.5rem` |
| maskClosable     | Whether can be closed by click mask        | `boolean`            | true      |
| title            | Custom title                               | `React.ReactNode`    | Title     |
| footer           | Custom footer                              | `React.ReactNode`    | -         |
| okText           | Custom the text of the confirmation button | `React.ReactNode`    | Ok        |
| cancelText       | Custom the text of the cancel button       | `React.ReactNode`    | Cancel    |
| open             | Whether to display                         | `boolean`            | -         |
| onOk             | Click the "Confirm" button callback        | `() => void`         | -         |
| onClose          | Callback when the modal is closed.         | `() => void`         | -         |
