</Code>

## API

### Slider Props

| 属性         | 描述         | 类型                      | 默认值 |
| ------------ | ------------ | ------------------------- | ------ |
| className    | 节点类名     | `string`                  | -      |
| defaultValue | 默认值       | `number`                  | -      |
| value        | 受控的值     | `number`                  | -      |
| max          | 最大值       | `number`                  | 100    |
| step         | 步长         | `number`                  | 1      |
| onChange     | 滑动时的回调 | `(value: number) => void` | -      |
