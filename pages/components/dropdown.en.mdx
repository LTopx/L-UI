## API

### Dropdown Props

| Name        | Description                       | Type                                | Default  |
| ----------- | --------------------------------- | ----------------------------------- | -------- |
| className   | Additional css class              | `string`                            | -        |
| disabled    | Whether is disabled               | `boolean`                           | -        |
| side        | Pop-up direction                  | `top` \| `bottom`                   | `bottom` |
| sideOffset  | Spacing of the pop-up direction   | `number`                            | 6        |
| align       | Alignment Direction               | `start` \| `center` \| `end`        | `center` |
| alignOffset | Alignment Direction Spacing       | `number`                            | -        |
| label       | Drop area title                   | `React.ReactNode`                   | -        |
| options     | Content of dropdown list          | `DropdownOption[]`                  | -        |
| selectable  | Whether can be selected           | `boolean`                           | -        |
| value       | The currently selected value.     | `string` \| `number`                | -        |
| onSelect    | Callback for selecting an option. | `(value: string \| number) => void` | -        |

### DropdownOption Props

| Name  | Description                            | Type              | Default |
| ----- | -------------------------------------- | ----------------- | ------- |
| label | Options Display Content                | `React.ReactNode` | -       |
| value | The value when the option is selected. | `string`          | -       |
| type  | Option type                            | `seperate`        | -       |
